Procédure installation json server: 

    - npm install json-server
    - Créer un db.json à la racine du projet
    - Rajouter le script suivant dans le package.json:
        "json-server": "json-server db.json"
    - lancer le json-server avec "npm run json-server"

POST: pour création d'une ressource

PUT: pour mettre à jour d'une ressource

GET: pour 'obtenir' une ressource

DELETE: pour supprimer une ressource
